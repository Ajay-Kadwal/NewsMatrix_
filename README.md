NewsMatrix is a streamlined, production-ready iOS application engineered to deliver real-time news content through a clean, responsive, and modern SwiftUI experience. The solution integrates external APIs, applies structured decoding, and leverages an MVVM-driven architecture to ensure scalability and maintainability.

Overview:
NewsMatrix fetches and displays curated headlines from a remote API, presents structured article details, and provides users with a frictionless reading workflow. The application emphasizes performance, clean code practices, and an enterprise-grade user experience.

Key Features
	•	Live News Feed
Fetches real-time articles with seamless asynchronous networking.
	•	SwiftUI Interface
Modern, adaptive UI components optimized for readability and performance.
	•	Article Detail View
Full-screen view displaying title, description, source, author, and image.
	•	API Integration
Clean URLSession implementation with modular request handling.
	•	Error Management
Structured fallback states, user-friendly loading and failure indicators.
	•	MVVM Architecture
Clear separation of concerns for scalable and maintainable development.
